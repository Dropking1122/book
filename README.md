# WebBook

WebBook adalah sebuah website sederhana untuk merayakan momen spesial bersama orang tercinta. Anda bisa menggunakannya untuk anniversary hubungan, ulang tahun pasangan, teman, atau keluarga, bahkan untuk memberi kejutan secara online.

## ✨ Fitur

- Tampilan website yang romantis, friendly, dan mudah diedit
- Bisa dikustomisasi sesuai kebutuhan dan keinginan pengguna
- Cocok untuk berbagai acara personal seperti anniversary dan ulang tahun
- Dapat di-host gratis menggunakan GitHub Pages atau layanan serupa

## 🚀 Demo

Kamu bisa lihat demo dengan meng-clone repository ini lalu jalankan secara lokal di browser, atau deploy ke GitHub Pages/fleksibel ke hosting statis.

## 📦 Cara Menggunakan

1. **Fork** repository ini ke akun GitHub milikmu.
2. **Clone** repositori:
   ```bash
   git clone https://github.com/Dropking1122/book.git
   ```
3. **Edit file HTML/CSS/JS** sesuai keinginan kamu (ubah kata-kata, gambar, dan tema).
4. **Commit & push** perubahan ke GitHub.
5. **Aktifkan GitHub Pages** pada repository untuk publikasi web secara online.

## 🛠️ Kustomisasi

- Semua konten bisa diedit pada file utama di direktori repo.
- Ubah nama, tanggal, pesan, gambar, warna, atau tambahkan elemen baru sesuai kreativitasmu.
